Crossway API

Api for offering SaaS applications as SOA's concepts, strongly recomended for Google AppEngine applications.

Welcome to Crossway API web page

Feel free to use the Crossway API to get easier provide your SaaS application as SOA's concept, a easy way to change the concept of your project, especially if you work with Google AppEngine. With few annotations you will see how you'll easily provide different processes for each client without changes in your frontend. See the Documentation and Examples.
